Simulation de mon application cliente android ChatApp'

requête HTTP POST - paramètres String

Je simule l'envoi d'une requête HTTP POST envoyée par mon application android ChatApp' :

http://serverWebChatAppBouchon/post_Request?pseudo="Samir"&contact="Toto"&message="T'es où Toto ?"

- le paramètre "pseudo" de l'URL contient l'émetteur du message.

- le paramètre "contact" de l'URL contient le récepteur du message.

- le paramètre "message" de l'URL contient le message.
